 Mars Photos App

An Android application built using Kotlin and Jetpack Compose that displays real images of Mars fetched from NASA’s API 🌌🛰️
-----
** Features
- View Mars photos in a beautiful grid layout
- Tap on any image to view full details
- Fetches real-time data from NASA API
- Smooth and responsive UI using Jetpack Compose
- Navigation between screens (Home → Detail)
- MVVM architecture for clean code structure
** Tech Stack
Kotlin
Jetpack Compose
MVVM Architecture
Retrofit (API calls)
Coil (Image loading)
Navigation Compose

⚙️ Setup Instructions
-----
Clone the repository
git clone https://github.com/rithvik4400/marsphotos.git
Open in Android Studio
Add Internet Permission in AndroidManifest.xml
<uses-permission android:name="android.permission.INTERNET"/>
Run the app on emulator or device
